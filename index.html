Network Performance Dashboard

A lightweight, browser-based tool for real-time monitoring of your network latency and stability.

This project provides a self-contained web application to help you visually track:

  • Live latency to key internet services (Cloudflare & Google).
  • A historical log of latency spikes and their duration.
  • A real-time chart showing latency trends over a configurable time window.
  • A command-line style feed of every ping result.

All settings, like ping frequency and spike sensitivity, are fully interactive.
